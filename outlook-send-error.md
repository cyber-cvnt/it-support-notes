# Outlook Cannot Send Emails

## Problem
A user is unable to send or reply to emails in Microsoft Outlook.

## Symptoms
- Emails remain stuck in the **Outbox**
- Error messages appear when sending
- Large attachments may fail to send

## Possible Causes
- Mailbox storage limit reached
- Large attachments
- Outlook offline mode
- Corrupted Outlook profile

## Troubleshooting Steps

### 1. Check Mailbox Size
Navigate to:

File → Tools → Mailbox Cleanup

Delete unnecessary emails if mailbox quota is exceeded.

### 2. Remove Large Attachments
Check Outbox for emails with large attachments and remove them.

### 3. Verify Outlook is Online
Ensure Outlook is not in **Work Offline Mode**.

Delete unnecessary emails if mailbox quota is exceeded.

### 2. Remove Large Attachments
Check Outbox for emails with large attachments and remove them.

### 3. Verify Outlook is Online
Ensure Outlook is not in **Work Offline Mode**.
Send/Receive → Work Offline
### 4. Restart Outlook
Close Outlook completely and reopen the application.

### 5. Repair Outlook Profile
Navigate to:

Control Panel → Mail → Show Profiles

Create a new profile if necessary.

## Resolution
After clearing mailbox storage and restarting Outlook, the user was able to send emails successfully.

## Lessons Learned
Mailbox quotas and large attachments are common causes of sending failures.
